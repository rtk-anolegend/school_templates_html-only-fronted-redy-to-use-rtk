# 🎓 School Website Frontend Template

A modern, responsive, and beginner-friendly school website template built using HTML, CSS, and JavaScript.

---

## 🚀 Why This Project Exists?

Nowadays, many website templates are sold at high prices 💸  
Because of this, beginners and students cannot explore real-world website designs.

This project is created to:
- Help students learn how real websites are built
- Provide free and open-source code
- Allow anyone to edit, experiment, and improve

💡 When I was learning, I also wanted ready code to modify and understand — this project is built with that same vision.

---

## 📱 How to Use on Android (Step-by-Step)

You can run and edit this project directly on your mobile using:

👉 **SPCK Editor App**

### Steps:
1. Download **SPCK Editor** from Play Store
2. Open the app
3. Click on "Open Folder" or "Import Project"
4. Paste or copy project files
5. Open `index.html`
6. Click on **Run ▶️**

🎉 Now your website will open in mobile browser!

---

## 💻 How to Use on PC

1. Download the project
2. Open in VS Code
3. Open `index.html`
4. Right-click → Open with Live Server
